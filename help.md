# Paper Reviewer 사용 가이드

**Paper Reviewer**는 연구자와 학생을 위한 AI 기반 논문 독해 도구입니다. 웹 브라우저에서 실행되지만, 내 컴퓨터의 폴더에 파일을 직접 저장하고 관리합니다.

## 🚀 시작하기

1. **프로그램 실행**: 바탕화면의 `start_browser.bat` 파일을 더블 클릭합니다.
2. **브라우저 열림**: 기본 웹 브라우저가 열리며 앱이 실행됩니다.
3. **파일 준비**: 
   - `paper-reader-data` 폴더가 자동으로 생성됩니다. 
   - 읽고 싶은 `.html` 또는 `.md` 파일을 탐색기에서 드래그하여 앱 화면에 놓으면 서재에 추가됩니다.

---

## 💡 주요 기능

### 1. 서재 관리 (Library)
- **파일 추가**: 파일을 화면에 드래그 앤 드롭하면 자동으로 라이브러리에 복사됩니다.
- **목록 보기**: 왼쪽 사이드바(`Alt+1`)에서 저장된 파일 목록을 볼 수 있습니다.

### 2. 읽기 및 주석
- **하이라이트**: 텍스트를 드래그하면 툴바가 나타납니다. 색상을 선택하여 중요 내용을 표시하세요.
- **메모**: 하이라이트된 부분을 클릭하거나 우클릭하여 개인 메모를 남길 수 있습니다.
- **북마크**: 문단 번호를 클릭하면 해당 위치가 저장됩니다. 나중에 해당 파일을 열 때 그 위치로 이동합니다.

### 3. AI 연구 도구 (Research Tools)
AI 기능을 사용하려면 **설정(`Alt+S`)**에서 API 키를 입력해야 합니다. (DeepSeek, Gemini, OpenAI 지원)

- **문단 요약**: 원하는 문단을 선택하고 `요약` 버튼을 누르면 AI가 핵심 내용을 정리해줍니다.
- **용어 설명**: 모르는 단어를 드래그하고 `설명(Explain)`을 선택하면 AI가 설명을 팝업으로 보여줍니다.
- **AI 토론**: 오른쪽 사이드바(`Alt+4`)를 열고 "AI와 토론하기"를 통해 논문의 내용에 대해 질문하거나 비판적 토론을 할 수 있습니다.

### 4. 설정 최적화
- **모델 할당**: 설정 메뉴 > 모델 탭에서 기능별로 다른 모델을 지정할 수 있습니다.
  - **빠른 설명/요약**: `Gemini 1.5 Flash` 또는 `DeepSeek Chat` 추천 (속도 빠름, 저렴함).
  - **깊은 토론/추론**: `DeepSeek Reasoner (R1)` 또는 `GPT-4o` 추천 (똑똑함).

## ⌨️ 단축키 목록

| 기능 | 단축키 | 설명 |
|------|--------|------|
| **서재 열기** | `Alt + 1` | 왼쪽 사이드바 토글 |
| **목차 보기** | `Alt + 2` | 문서 목차(TOC) 패널 |
| **하이라이트** | `Alt + 3` | 하이라이트 목록 패널 |
| **AI 에이전트** | `Alt + 4` | 오른쪽 연구 에이전트 패널 |
| **노트북** | `Alt + 5` | 오른쪽 빠른 메모장 |
| **설정** | `Alt + S` | 설정 창 열기 |
| **한/영 모드** | `Alt + T` | (번역 기능 활성화 시) 언어 전환 |

## ❓ 자주 묻는 질문

**Q. 데이터는 어디에 저장되나요?**
A. 프로젝트 폴더 내 `paper-reader-data` 폴더에 저장됩니다. 앱을 끄더라도 데이터는 안전하게 유지됩니다.

**Q. 수식이 깨져 보입니다.**
A. 원본 HTML 파일이 MathML 또는 LaTeX를 포함하고 있는지 확인하세요. Paper Reviewer는 웹 표준 수식 렌더링을 지원합니다.

**Q. 툴바가 자꾸 떠서 불편해요.**
A. 텍스트를 짧게 클릭하면 뜨지 않도록 개선되었습니다. 5글자 이상 드래그해야 툴바가 나타납니다.

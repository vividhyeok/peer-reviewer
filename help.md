# 📖 Peer Reviewer 사용 가이드

**Peer Reviewer**는 연구자와 학생들이 논문을 더 빠르고 깊이 있게 이해할 수 있도록 돕는 AI 기반 논문 분석 도구입니다. 
`Immersive Translate`로 번역된 HTML 파일을 활용하면 비용 효율적으로 한/영 대조 독서가 가능합니다.

---

## 🚀 빠른 시작 (워크플로우)

### 1단계: 논문 준비 (Browser)
1.  브라우저(Chrome/Edge)에서 읽고 싶은 **영어 논문 PDF**를 엽니다.
2.  `Immersive Translate` 확장 프로그램을 사용하여 **[번역]**을 실행합니다.
3.  번역 완료 후 우측 하단 **[기타] -> [단일 HTML 파일로 내보내기]** (Bilingual HTML)를 선택하여 저장합니다.

### 2단계: 문서 열기 (App)
1.  앱 좌측 **[라이브러리]** 탭에서 **[+]** 버튼을 누르거나 파일을 드래그하여 HTML 파일을 엽니다.
2.  문서가 로드되면 AI가 자동으로 구조를 분석합니다.

---

## 🛠 주요 기능 상세 가이드

### 1. 듀얼 리더 & 뷰어
- **한영 대조**: 왼쪽에는 원문(영어), 오른쪽에는 한글 번역이 문단별로 정렬되어 표시됩니다.
- **툴팁 모드**: **설정 > [한글 우선 보기]**를 켜면 기본적으로 한글만 보이고, 마우스를 올렸을 때 원문이 툴팁으로 나타납니다.
- **이미지 뷰어**: 본문의 이미지를 클릭하면 확대되며, **[AI 설명]** 버튼으로 시각적 분석을 요청할 수 있습니다.

### 2. 하이라이트 & 메모
- 텍스트를 드래그하여 선택하면 툴바가 나타납니다.
- **형광펜**: 노랑, 빨강, 파랑 등 색상별로 중요도를 표시하세요.
- **메모**: 하이라이트된 텍스트에 개인적인 노트를 남길 수 있습니다.

### 3. 상단 툴바 (AI 도구)
선택한 텍스트에 대해 즉각적인 AI 분석을 수행합니다.
- **설명 (Explain)**: 어려운 개념이나 문장을 쉽게 풀이합니다.
- **요약 (Summarize)**: 긴 섹션을 핵심만 요약합니다.
- **비평 (Critique)**: 논리의 허점이나 방법론적 한계를 비판적으로 분석합니다.
- **쉽게 풀이 (Simplify)**: 복잡한 학술 용어를 일반인 수준으로 풀어줍니다.
- **한영 전환**: 뷰어 모드를 [한글 우선] / [영어 우선]으로 즉시 전환합니다.

### 4. 리서치 에이전트 (Research Agent)
우측 사이드바의 **[AI 에이전트]** 탭에서 심층 연구를 수행하세요.
- **심층 채팅**: 단순 문답을 넘어, 에이전트가 "계획(Plan)"을 수립하여 답변합니다.
  - 예: "이 알고리즘의 시간 복잡도를 증명와 함께 설명해줘."
- **자동 브리핑**: 문서를 열면 **[요약]** 탭에서 3줄 요약, 연구 목표, 방법론, 한계점을 자동으로 정리해줍니다.

### 5. AI 토론 (Discussion)
- 특정 문단을 선택하고 **[AI 질문]**을 클릭하면 대화형 모달이 열립니다.
- "이전 연구와의 차이점은?", "핵심 가정은?" 등의 추천 질문으로 빠르게 문맥을 파악할 수 있습니다.

### 6. 스마트 내보내기 (Export)
- 상단 **[내보내기]** 버튼을 클릭하세요.
- 하이라이트, 메모, AI와의 대화 내용을 **Markdown (Obsidian 호환)** 형식으로 추출하여 나만의 연구 노트를 구축할 수 있습니다.

---

## ⚙️ 설정 (Settings)

우측 하단 톱니바퀴 아이콘을 눌러 환경을 설정하세요.

- **API 키**: OpenAI, Anthropic, Gemini, DeepSeek 등의 키를 입력합니다. (로컬 저장소에만 저장됨)
- **모델 할당**: 요약은 빠르고 저렴한 모델(Haiku, Flash), 심층 분석은 고성능 모델(GPT-4, Opus)로 지정하여 비용을 최적화하세요.
- **단축키**: `Alt + S` (요약), `Alt + E` (설명) 등 편리한 단축키를 확인하고 변경할 수 있습니다.
- **저장소**: 브라우저 캐시 삭제가 걱정된다면 **[로컬 폴더 선택]**을 통해 데이터를 내 PC의 폴더에 영구 저장하세요.

## ❓ 자주 묻는 질문

**Q. 영어가 나오다가 중간에 한글이 섞여 나와요.**
A. 원본 HTML 구조 문제일 수 있습니다. `Immersive Translate` 설정에서 '문단 단위 번역'을 활성화한 후 다시 내보내 보세요. 또한 리더기 설정에서 **[기본 번역 모드]**를 변경해 보시기 바랍니다.

**Q. 데이터는 어디에 저장되나요?**
A. 기본적으로 브라우저의 IndexedDB에 저장됩니다. **설정 > 저장소** 탭에서 로컬 폴더를 연결하면 내 컴퓨터의 파일 시스템에 직접 저장할 수 있어 더욱 안전합니다.
